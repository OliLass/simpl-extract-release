# SimplExtract — Téléchargements

Application pour les formateurs Simplon : téléchargement en lot des rendus étudiants et des bilans PDF depuis Simplonline.

## Télécharger

Rendez-vous dans la section [**Releases**](https://github.com/OliLass/simpl-extract-release/releases) pour télécharger la dernière version.

| Plateforme | Fichier |
|---|---|
| macOS (Apple Silicon + Intel) | `SimplExtract_x.x.x_universal.dmg` |
| Windows (installeur) | `SimplExtract_x.x.x_x64-setup.exe` |
| Windows (MSI) | `SimplExtract_x.x.x_x64_en-US.msi` |
| Windows (portable) | `SimplExtract-portable.exe` |

## Fonctionnalités

- Téléchargement en lot des rendus (fichiers + dépôts GitHub en ZIP)
- Génération d'un énoncé `enonce.html` dans chaque dossier de brief
- Téléchargement des bilans de formation PDF
- Thème clair / sombre
